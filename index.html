Instruções para Exclusão de Dados

Se você deseja excluir seus dados da nossa aplicação, siga as instruções abaixo:

  1. Abra o aplicativo em seu dispositivo.
  2. Vá para o menu Configurações ou Perfil.
  3. Selecione a opção Excluir Conta ou Excluir Dados.
  4. Confirme a exclusão. Todos os seus dados serão removidos de forma permanente.

Se tiver dúvidas, envie um e-mail para: seuemail@exemplo.com
